
这是一个关于一个springCloud2的demo,
集成了 eurekaServer/feign/zuul/mybatis
适合于初学者参考。
统一网关入口：http://localhost:8799/用于访问注册的和种服务。
我会逐一更新。

（1）这时微服务必不可少的4个服务：

（2）加上了mybatis后的服务
能访问数据库的服务temp-mybatis-service
(3) mybatis generator,定制插件，生成xml,mapper,service,controller
(4) updating....
